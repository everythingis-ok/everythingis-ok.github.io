ctf介绍

CTF入门

可选方向
web
reverse(二进制方向)
pwn(二进制方向)

必选
crypto(密码学)
misc（杂项）

CTF竞赛 攻防模式
每队运维3-4个存在漏洞的网络服务 找到漏洞先修复 利用漏洞编写攻击其他队伍

web方向（网络攻防的重要入口）

  • 重点关注：网站的安全
  • web应用的常见漏洞：
    • SOL注入
    • 文件上传漏洞
    • 文件包含漏洞
    • 反序列化漏洞

Web方向-前端

  • 前台的显示，外观以及前台的简单交互
  • 代码实现
    • JaveScript
    • HTML
    • CSS

cypto方向(密文的安全)

  • 重点关注：密文的安全
  • 密码学常见类型
    • 凯撒密码/维吉尼亚密码
    • DES/AES/RSA/…

Misc方向（现实中广泛应用的安全技术）

  • Misc题目常见类型：
    - 压缩包破解/隐写/流量分析/…

Reverse方向（程序破解的基础）

  • 重点关注：程序运行的逻辑
  • 逆向还原程序的编程逻辑
    • 脱壳/加壳
    • 加密算法程序
    • IDA Pro

Pwn方向（二进制漏洞利用的基础）

  • 重点关注：二进制漏洞的利用
  • 利用程序的漏洞实现攻击
    • 栈溢出
    • 格式化字符串溢出
    • 堆溢出
